 Como utlizar o assistente

O assistente do Sistema WFA Web pode ser acionado de qualquer parte do sistema, clicando sobre o link AJUDA localizado na parte superior direita da tela, ou através do menu AJUDA.
Para obter ajuda sobre um tópico, basta clicar sobre a frase. Sugere-se que este assistente, ao ser lido, seja acompanhado pela navegação no Sistema WFA Web, para um maior entendimento.

 Efetuando login no sistema

Ao iniciar o sistema, a primeira etapa é efetuarmos login no sistema, conforme tela abaixo:
1 - Login: deve ser informado login de acesso ao sistema. O login de acesso será normalmente o código de sua loja junto a Arezzo;
2 - Senha: deve ser informada a senha de acesso ao sistema;
3 - Esqueci minha senha: esta opção deverá ser utilizada quando o usuário não lembrar sua senha de acesso ao sistema;
Voltar ao início

 Tela principal do sistema

Ao efetuar login no sistema, o usuário é direcionado para tela principal do sistema, conforme demonstrada abaixo:
1 - USUÁRIO LOGADO: nome do usuário logado no sistema;
2 - logout: clique nesta opção para efetuar sair do sistema e/ou trocar de usuário;
3 - MENUS HORIZONTAIS: menus de acesso as diversas funcionalidades do sistema;

 Menu - Cliente - Inclusão de reclamação

Esta opção do menu é utilizada para inclusão de reclamações. Nela serão inseridas todas as informações referente ao defeito encontrado no produto, assim como dados referentes a NF de origem, fabricante do produto, fotos do produto, etc.

O preenchimento desta tela é muito importante, pois a partir desses dados que a Retaguarda - Arezzo irá realizar a análise do problema. Os campos em amarelo serão preenchidos automaticamente pelo sistema.
 
1 - CATEGORIA Selecione a categoria de produto a ser aberta a reclamação.
2 - CÓDIGO DO CLIENTE Selecione o código do cliente a que pertence a reclamação.
3 - SOLICITANTE Nome do solicitante da reclamação. Por default o sistema já insere o nome do usuário logado neste campo.
4 - MOTIVO DA SOLICITAÇÃO Descreva o motivo da solicitação da análise.
5 - TIPO RECLAMAÇÃO Assinalar DEFEITO CONSUMIDOR quando o problema foi constatado após a venda ao cliente. Assinalar DEFEITO LOJA quando o problema foi constatado antes mesmo de ser efetuada a venda do produto.
6 - NOME DO CONSUMIDOR Quando for DEFEITO CONSUMIDOR preencher com o nome do cliente. Caso seja DEFEITO LOJA, informar o nome da loja
7 - FONE DO CONSUMIDOR Quando for DEFEITO CONSUMIDOR preencher com o telefone do cliente. Caso seja DEFEITO LOJA, informar o telefone da loja
8 - REFERÊNCIA Informe a referência do produto a ser análisado. Ex.: 0180964100010002 (a referência sempre deve ser informada completa, ou seja, 16 caracteres).
9 - QTDE

Informar a respectiva quantidade a ser analisada em cada número do par. Ex.: 1 n.o 33 + 3 n.o 35
Obs.: este ítem é somente para categoria CALÇADOS

10 - FABRICANTE A listagem de fabricantes será preenchida a partir do momento em que o usuário clicar em PESQUISAR DADOS DA NF (15) com base na referência do produto. A partir daí, o usuário deve selecionar o fabricante conforme código que consta na sola do calçado.
11 - N° NF ORIGEM Informe o número da nota fiscal de origem, caso possua. Em caso de não possuí-la, preencha a referência e clique em PESQUISAR DADOS DA NF. O sistema exibirá ao usuário notas fiscais faturadas para o código de cliente + referência informada. Em caso de encontrar apenas uma nota fiscal faturada para o código de cliente + referência, o sistema irá preencher automaticamente os campos: nota fiscal de origem, emitente, data da nf, valor unitário. Caso encontre mais de uma nota fiscal, o sistema irá lhe exibir uma listagem de todas as notas fiscais, com suas respectivas datas de emissão e emitente possibilitando ao usuário selecionar a nota fiscal desejada.
12 - PESQUISAR DADOS NF

Quando informado número da nota fiscal de origem e referênica: clique neste botão para validar os dados informados. Caso o sistema valide os dados, o sistema preencherá os campos em amarelo automaticamente.

Quando informado apenas a referência: clique neste botão para que o sistema localize todas as notas fiscais de origem emitidas para o código do cliente selecionado + referência informada. Caso o sistema localize apenas uma nota fiscal, os campos em amarelo serão preenchidos automaticamente. Caso localize mais de uma nota fiscal, o sistema irá lhe exibir uma listagem de todas as notas fiscais, com suas respectivas datas de emissão e emitente possibilitando ao usuário selecionar a nota fiscal desejada.
13 - FOTO PRODUTO O usuário deverá selecionar foto do produto. A foto do produto deverá conter foto do par completo do calçado reclamado. O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
14 - FOTO SOLA O usuário deverá selecionar foto da sola. A foto da sola deverá conter foto da(s) sola(s) do(s) pé(s) reclamado(s). O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
15 - FOTO DEFEITO O usuário deverá selecionar foto do defeito. A foto do defeito deverá conter foto nítida do defeito reclamado. O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
Clique no botão GRAVAR para salvar os dados informados. Após gravar os dados da reclamação, o sistema gerará automaticamente ETIQUETAS que deverão ser recortadas e coladas em cada caixa individual do(s) par(es) reclamado(s). Veja IMPRESSÃO DE ETIQUETA INDIVIDUAL
Clique no botão CANCELAR para cancelar a digitação dos dados informados e retornar à listagem de clientes.
