 Como utlizar o assistente

O assistente do Sistema WFA Web pode ser acionado de qualquer parte do sistema, clicando sobre o link AJUDA localizado na parte superior direita da tela, ou através do menu AJUDA.
Para obter ajuda sobre um tópico, basta clicar sobre a frase. Sugere-se que este assistente, ao ser lido, seja acompanhado pela navegação no Sistema WFA Web, para um maior entendimento.
Voltar ao início

 Efetuando login no sistema

Ao iniciar o sistema, a primeira etapa é efetuarmos login no sistema, conforme tela abaixo:
1 - Login: deve ser informado login de acesso ao sistema. O login de acesso será normalmente o código de sua loja junto a Arezzo;
2 - Senha: deve ser informada a senha de acesso ao sistema;
3 - Esqueci minha senha: esta opção deverá ser utilizada quando o usuário não lembrar sua senha de acesso ao sistema;
Voltar ao início

 Tela principal do sistema

Ao efetuar login no sistema, o usuário é direcionado para tela principal do sistema, conforme demonstrada abaixo:
1 - USUÁRIO LOGADO: nome do usuário logado no sistema;
2 - logout: clique nesta opção para efetuar sair do sistema e/ou trocar de usuário;
3 - MENUS HORIZONTAIS: menus de acesso as diversas funcionalidades do sistema;

 Menu - Cliente - Inclusão de reclamação

Esta opção do menu é utilizada para inclusão de reclamações. Nela serão inseridas todas as informações referente ao defeito encontrado no calçado, assim como dados referentes a NF de origem, fabricante do calçado, número do par reclamado, fotos do produto, sola e defeito, etc.

O preenchimento desta tela é muito importante, pois a partir desses dados que a Retaguarda - Arezzo irá realizar a análise do problema. Os campos em amarelo serão preenchidos automaticamente pelo sistema.
 
 
1 - CÓDIGO DO CLIENTE Selecione o código do cliente a que pertence a reclamação.
2 - SOLICITANTE Nome do solicitante da reclamação. Por default o sistema já insere o nome do usuário logado neste campo.
3 - MOTIVO DA SOLICITAÇÃO Descreva o motivo da solicitação da análise.
4 - EMITENTE Código do emitente da nota fiscal de faturamento. Este campo é preenchido automaticamente com base na nota fiscal de origem.
5 - REFERÊNCIA Informe a referência do calçado a ser análisado. Ex.: 0180964100010002 (a referência sempre deve ser informada completa, ou seja, 16 caracteres).
6 - N° DO PAR RECLAMADO Informe o número do par que está sendo reclamado.
7 - FABRICANTE A listagem de fabricantes será preenchida a partir do momento em que o usuário clicar em PESQUISAR DADOS DA NF (15) com base na referência do calçado. A partir daí, o usuário deve selecionar o fabricante conforme código que consta na sola do calçado.
8 - COLEÇÃO Coleção referente à referência informada. Este campo é preenchido automaticamente.
9 - QUANTIDADE RECLAMADA Informe a quantidade de pares que está sendo reclamada.
10 - N° NF ORIGEM Informe o número da nota fiscal de origem, caso possua. Em caso de não possuí-la, preencha a referência e clique em PESQUISAR DADOS DA NF. O sistema exibirá ao usuário notas fiscais faturadas para o código de cliente + referência informada. Em caso de encontrar apenas uma nota fiscal faturada para o código de cliente + referência, o sistema irá preencher automaticamente os campos: nota fiscal de origem, emitente, data da nf, valor unitário. Caso encontre mais de uma nota fiscal, o sistema irá lhe exibir uma listagem de todas as notas fiscais, com suas respectivas datas de emissão e emitente possibilitando ao usuário selecionar a nota fiscal desejada.
11 - N° SÉRIE Número de série da NF de origem. Campo preenchido automaticamente.
12 - DATA NF Data de emissão da NF de origem. Campo preenchido automaticamente.
13 - VALOR UNITÁRIO Valor unitário faturado na NF de origem. Campo preenchido automaticamente.
14 - VALOR TOTAL Valor total calculado automaticamente, com base na quantidade de pares reclamados. Fórmula cálculo: QUANTIDADE RECLAMADA * VALOR UNITÁRIO. Campo preenchido automaticamente.
15 - PESQUISAR DADOS NF

Quando informado número da nota fiscal de origem e referênica: clique neste botão para validar os dados informados. Caso o sistema valide os dados, o sistema preencherá os campos em amarelo automaticamente.

Quando informado apenas a referência: clique neste botão para que o sistema localize todas as notas fiscais de origem emitidas para o código do cliente selecionado + referência informada. Caso o sistema localize apenas uma nota fiscal, os campos em amarelo serão preenchidos automaticamente. Caso localize mais de uma nota fiscal, o sistema irá lhe exibir uma listagem de todas as notas fiscais, com suas respectivas datas de emissão e emitente possibilitando ao usuário selecionar a nota fiscal desejada.
16 - FOTO PRODUTO O usuário deverá selecionar foto do produto. A foto do produto deverá conter foto do par completo do calçado reclamado. O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
17 - FOTO SOLA O usuário deverá selecionar foto da sola. A foto da sola deverá conter foto da(s) sola(s) do(s) pé(s) reclamado(s). O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
18 - FOTO DEFEITO O usuário deverá selecionar foto do defeito. A foto do defeito deverá conter foto nítida do defeito reclamado. O tamanho máximo da foto deverá ser de 100Kb. Para selecionar a foto, clique em PESQUISAR.
Clique no botão GRAVAR para salvar os dados informados.
Clique no botão CANCELAR para cancelar a digitação dos dados informados e retornar à listagem de clientes.
